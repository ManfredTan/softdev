Google Page Rank Algorithm


What is it?

    - PageRank is one of many factors Google uses in ranking the pages that appear in a search
    - Assigns as number (1-10) to each page based on its unique algorithm
    - Utilizes a algorithm that is based on a damping factor and the sum of the incoming PageRank scores

What does it value?

    - Older websites
    - The number of links that direct to the given webpage
    - The frequency of how often the website is searched

How Does it Work?

Imagine four independent websites. In order to determine which website should be the first result after a google search, we need to rank them. So, we give each website a "point" and allow them to vote for which website is the highest.


These were the results:
A chose B
B chose C and D
C chose A and D
D chose A and B

Just by looking at the number of points that each website received, B should be first and C should be last. But we also need to figure out if A or D should be higher. So the question we now ask is: On which sites do we spend the most time on? What percent of our time would be spent on each site? Thus, the "points" become links that lead to the other websites.

To answer this question, we start at a random site. We assume that each site has an equal probability of being chosen, so we give them a 0.25 probability. We multiply our matrix of values with the 0.25 matrix to get the results.

These results make sense. Let's say we start on website B. B chose both C and D, so they both have a (0.25) x (0.5) probability of getting there from website B. Only website B links to C, so the final probability for C is 0.125. D, on the other hand, is also linked from C. If we start on C, there is a 0.125 chance of going to D. Thus we add 0.125 from B to D and 0.125 from C to D and the final probability for website D is 0.25. If we repeat this for all websites, the results should be the same as in the matrix.

But this is only for one round of clicks. We want to know what website would be the highest after many clicks. If we multiply the original matrix with the new results over and over, we get:

A(A*R0) . . .lim n inf A^n R0

We can see that website B is the highest and website C is the lowest. We also see that website D is the second highest. This makes sense because while A and D both have 2 links going to the pages, D is linked by B, which would be considered a "reputable source". Thus D gets more points and ranks higher than A.

Obviously this is a very simple example, and doesn't account for other factors like new web pages and spam pages, but the general idea is captured. The real algorithm is only known by the google engineers themselves.
